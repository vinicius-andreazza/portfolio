// Sobre Mim

Engenheiro de Software

Minha trajetória começou na formação técnica do SENAI, onde desenvolvi uma visão orientada à lógica, análise de processos e solução de problemas complexos. Esse ambiente despertou meu interesse por sistemas robustos e pela forma como diferentes componentes trabalham em conjunto para entregar resultados confiáveis.

Ao migrar para o desenvolvimento de software, encontrei no ecossistema Java uma filosofia semelhante à engenharia industrial: componentes bem definidos, padrões sólidos e foco em estabilidade. Desde então, venho aprofundando meus conhecimentos em desenvolvimento backend, arquitetura de software, bancos de dados relacionais, APIs REST e microsserviços.

Acredito que um bom desenvolvedor backend não apenas escreve código, mas projeta estruturas capazes de suportar crescimento, mudanças e altas cargas de trabalho sem comprometer a qualidade do sistema.
